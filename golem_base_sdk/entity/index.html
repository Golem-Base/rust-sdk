Module for GolemBase entities and data types. Defines core types such as annotations, hashes, and entity representations.

Structs§

  • Annotation
    A generic key-value pair structure for entity annotations. Used for both string and numeric metadata attached to entities.
  • Create
    Type representing a create transaction in GolemBase. Used to define new entities, including their data, BTL, and annotations.
  • DeleteResult
    Represents the result of deleting an entity. Contains the key of the deleted entity.
  • Entity
    Represents an entity with data, BTL, and annotations. Used for reading entity state from the chain.
  • EntityResult
    Represents the result of creating or updating an entity. Contains the entity key and its expiration block.
  • Extend
    Type representing an extend transaction in GolemBase. Used to extend the BTL of an entity by a number of blocks.
  • ExtendResult
    Represents the result of extending an entity’s BTL. Contains the entity key, old expiration block, and new expiration block.
  • GolemBaseTransaction
    Type representing a transaction in GolemBase, including creates, updates, deletes, and extensions. Used as the main payload for submitting entity changes to the chain.
  • Update
    Type representing an update transaction in GolemBase. Used to update existing entities, including their data, BTL, and annotations.
